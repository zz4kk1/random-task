  Random Task Generator (Генератор случайных задач)

  Автор
Салихов Камиль


  Описание
Графическое приложение на Python для генерации случайных задач с возможностью фильтрации по категориям, добавления пользовательских задач и сохранения истории в формате JSON.

  Основные функции:
-  Генерация случайной задачи из встроенного или пользовательского списка
-  Фильтрация по типу: `учёба`, `спорт`, `работа`, `другое`
-  Отображение истории сгенерированных задач с временными метками
-  Автоматическое сохранение и загрузка данных между запусками
-  Валидация ввода при добавлении новых задач

  Как запустить
1. Убедитесь, что у вас установлен `Python 3.8+`
2. Скачать репозиторий, определить все папки из проекта в какую то одну и открыть ее через любое IDE
